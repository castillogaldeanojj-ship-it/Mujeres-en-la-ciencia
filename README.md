# Mujeres-en-la-ciencia
El presente documento recoge una propuesta didáctica enmarcada en el proyecto “Mujeres en la ciencia”, dirigida al alumnado de 1º de ESO en el área de Inglés. A lo largo de esta secuencia de aprendizaje, el alumnado se acerca al conocimiento de referentes femeninos en el ámbito científico.
